# TaskBootstrab

A simple, responsive portfolio landing page built with **Bootstrap 5**.

## Preview
- Open `index.html` in your browser.

## Tech Stack
- HTML5
- Bootstrap 5 (CDN)

## Getting Started (Local)
1. Clone the repo:
   ```bash
   git clone <your-repo-url>
   ```
2. Open the project folder.
3. Double-click `index.html` (or open it with VS Code + Live Server).

## Project Structure
- `index.html` — The full page markup and styling (includes small inline CSS).

## Customize
- Update your name and intro text in the **Home** section.
- Replace placeholder images by editing the `<img src="...">` URLs in `index.html`.
- Adjust colors and layout via Bootstrap classes and the inline styles.

## Deploy (GitHub Pages)
1. Push this repo to GitHub.
2. Go to **Settings → Pages**.
3. Under **Build and deployment**:
   - Source: **Deploy from a branch**
   - Branch: **main** (or **master**) / **root**
4. Save, then wait for the Pages URL.
